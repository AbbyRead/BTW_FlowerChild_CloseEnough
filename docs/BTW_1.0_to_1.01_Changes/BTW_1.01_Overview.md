## **BTW 1.01 Changes**

- Added ModLoader support  
- Added **Iron Pressure Plates**  
- Added wood and iron versions of the **Minecart Pressure Plate**  
  - Stone pressure plates from previous versions automatically convert to wood to preserve functionality

---

## All BTW 1.01 Recipes

### Cement Bucket

```plaintext
  [   |   |   ]
  [ S | W | G ]
  [   | B |   ]
  
  S = Slimeball
  G = Gravel
  W = Water Bucket
  B = Bucket
```

### Light Block

Note: Called "Light Bulb" in BTW 1.0

```plaintext
  [   | G |   ]
  [ G | L | G ]
  [   | R |   ]
  
  G = Glass
  L = Glowstone Dust
  R = Redstone
```

### "Minecart Booster" Block

```plaintext
  [ I | S | I ]
  [ I |   | I ]
  [ I | R | I ]
  
  I = Iron Ingot
  R = Redstone
  S = Soul Sand
```

### "Minecart Plate" Block (Stone)

Note: Was called "Minecart Pressure Plate" in BTW 1.0

```plaintext

  [ I | R | I ]
```
